# Running tests

```bash
./test.sh
```

# Executing

```bash
echo -n "()()" | ./run.sh
```

or

```bash
./run.sh
```

and then, provide the value by hand

## Disclaimer

It works only with parenthesis intentionally and it have a max size input of 50 bytes.

It returns `0` when successfully balanced and `1` when it's not.
